# KoC-Scripts
Various automated scripts for text based MMORPG 'KingsOfChaos'.

Current tools: 
               
               Auto Banker
               Auto Wizard
               Sell Catcher
               Auto Attacker (incomplete/not stable)

Written in Python 3.

Requires Beautiful Soup 4 and Selenium modules.
